Configuration and operations of general output pin

The function init_pin_as_general_output(Uint16) configures a GPIO to a general output pin with two possible states: HIGH (3,3 V) and LOW (0 V).
Via functions set_general_output_pin_high(Uint16) and set_general_output_pin_low(Uint16) the state of the respective pin can be controlled.
Possible GPIOs as input parameter are defined in global_defines.h

Code example

void main(void){
int result = 0;
 result = init_pin_as_general_output(GPIO0); //Configures GPIO0 as general output pin. Gives back result of operation
 }
